# [Burgr](burgr.herokuapp.com)

Find the best burger restaurants in Boston with Burgr! Burgr let's you can create an account, upload a profile picture, write your own reviews, and vote on other reviews you see!

## Technology
* Ruby on Rails
* React.js
* AWS
* Mailgunner
* Carrierwave
* Heroku
